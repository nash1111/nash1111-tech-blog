## 手順
[feed](https://www.npmjs.com/package/feed)を追加します
```
bun add feed
```
feedにrss2,atom1,json1が生えているため,feedをnewしてから
```
feed.rss2()
feed.atom1()
feed.json1()
```
の結果をそれぞれ```rss.xml,feed.json,atom.xml```に書くようにしました。

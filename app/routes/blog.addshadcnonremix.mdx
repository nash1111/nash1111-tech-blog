## 手順
tailwindとshadcn/uiを順に入れれば大丈夫でした
### tailwindの導入
tailwindがないとshadcn initの時エラーになるので、先にtailwindを入れます
[Install Tailwind CSS with Remix](https://tailwindcss.com/docs/guides/remix)に従えば問題ありませんでした
```
bun add tailwindcss postcss autoprefixer
npx tailwindcss init --ts -p
// tailwind.config.tsをドキュメント通り編集
// tailwind.cssをドキュメント通り編集
// root.tsxをドキュメント通り編集
```

こちらも[公式](https://ui.shadcn.com/docs/installation/remix)に従えば問題ありませんでした
